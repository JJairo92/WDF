Element Definitions

h1
A level one heading element, used to define the most important heading.
<h1>

This is a level one heading example.

</h1>

h2
A level two heading element, used to define the second most important heading.
<h2>

This is a level two heading example.

</h2>

h3
A level three heading element, used to define the third most important heading.
<h3>

This is a level three heading example.

</h3>

p
A paragraph element, used for making paragraphs.
<p>This is a paragraph example.</p>

ul
An unordered list (bullet point) element, used for listing items in unordered bullet point format.
<ul>
  • This is a bullet point example (must be used with the li element).
</ul>

ol
An ordered list (numbered list) element, used for listing items in ordered numberical format.
<ol>
  1. This is a numbered list example (must be used with the li element).
</ol>

li
An individual item or point in a list element, used to separate each item in the list.
  • <li>This is a list item example.</li>
