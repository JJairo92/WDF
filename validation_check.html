Websites I Validated

Kingdom Hearts Insider

  • www.khinsider.com

Website didn't pass validation in both of the HTML validators

HTML Lint Validator

In the HTML Lint Validator there was one error; an illegal closing tag.

</p> was used instead of </div>

The developer should just switch the </p> closing tag to </div>

nu Validator

In the nu Validator there were 103 errors; these include warnings and errors.

Some of the errors included XML attributes not allowed in HTML, element img required the alt attribute, the ampersand was used in an URL.

XML attributes should not be used, they can be taken out and replaced with HTML attributes. Also, the alt attribute should be added to the img elements, finally & entity should be used instead of the actual ampersand.

League of Legends

  • na.leagueoflegends.com

Website didn't pass validation in both of the HTML validators

HTML Lint Validator

In the HTML Lint Validator there were 11 errors; they were all about not closing empty tags.

There were no closing tags for 10 img empty tags, and for one hr empty tag.

The developer should just add the closing tags.

nu Validator

In the nu Validator there were 49 errors.

Some of the errors included element img required the alt attribute, the element h2 is not allowed as a child of the span element, the element style is not allowed as a child of div element.

The alt attribute should be added to the img elements, and not using the mentioned elements as children would also be a way to fix some of the errors.

YouTube

  • www.youtube.com

Website didn't pass validation in both of the HTML validators

HTML Lint Validator

In the HTML Lint Validator there were 216 errors.

They were mainly about not closing empty tags, and missing whitespace after closing a quote.

The developer should just add the closing tags, and add the whitespace after closing a quote.

nu Validator

In the nu Validator there were 59 errors.

Some of the errors included element img required the alt attribute, and using the ampersand instead of using the entity.

The alt attribute should be added to the img elements, and & entity should be used instead of the actual ampersand.

Wikipedia English Home Page

  • en.wikipedia.org/wiki/Main_Page

Website passed one of the HTML validators

Wikipedia English Main Page

In the HTML Lint Validator there were no errors.

nu Validator

In the nu Validator there were 15 errors.

Some of the errors included srcset attribute is not allowed in element img, and the center element is obsolete.

The srcset attribute should be taken out, and the developer should get used to not adding presentation to HTML; it should be used in CSS instead.
